### Pokemon Society

## Build project
```shell
npm run build
```

## Run server with nodemon
```shell
npm run dev
```

## Run server production build
```shell
npm run start
```

## .env

```env
LOG_LEVEL=<error | warn | info | debug>
PORT=<server port>
MYSQL_HOST=<localhost>
MYSQL_DB=<database name>
MYSQL_USER=<mysql user with right privileges>
MYSQL_PASSWD=<mysql user password>
```
